## How to use

Initial setup

```bash
git clone https://github.com/gentksb-kinto/dotfiles.git
./dotfiles/install/init.sh
./dotfiles/install/install.sh
```

Update or reconfigure setup

```bash
git clone https://github.com/gentksb-kinto/dotfiles.git
./dotfiles/install/install.sh
```
